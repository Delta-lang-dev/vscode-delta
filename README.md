# Δ Delta — VS Code Extension

The official VS Code extension for the [Delta Language](https://github.com/Delta-lang-dev/Delta).

## Features

- Syntax highlighting for `.delta` files
- Real-time diagnostics (errors and warnings as you type)
- Autocomplete for all Delta constructs
- Hover documentation for keywords
- Works with any LSP-compatible editor

## Install

Search for **Delta Language** in the VS Code Extensions marketplace, or install via the command line:

```bash
code --install-extension delta-lang.delta-lang
```

## Supported Constructs

`patch` · `fix` · `intent` · `migrate` · `guard` · `trace` · `apply`

## Requirements

- VS Code 1.85.0 or higher
- [Delta CLI](https://github.com/Delta-lang-dev/Delta) installed globally
